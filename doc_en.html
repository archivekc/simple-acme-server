How to get a certificate with this server

Trust the acme server on your station

you need to import the CA certificate on your station.

Linux server

sudo wget --no-check-certificate -O /usr/local/share/ca-certificates/{{.Domain}}.crt "{{.ServerURL}}/cert/ca/pem"
sudo update-ca-certificates

Windows

Download the CA certificate and add it to the CA truststore as a binary certificate.

Request a certificate to the acme server

Using lego (https://github.com/xenolf/lego)

With the command line utility lego you can create a new certificate for the name my.example.org simply with the following command:

lego -d my.example.org -s {{.ServerURL}}/directory -m admin@my.example.org run

Where my.example.org is your domain name and admin@my.example.org your email adress.
